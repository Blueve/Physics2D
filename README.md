Physics2D
=========

A physics engine build via C# with WPF Demo &amp; Unit Test.

一个使用C#编写的2D物理引擎，项目包含了

  - Physics Engine 物理引擎
  - Unit Test 单元测试
  - WPF Demo 基于WPF绘制的简单Demo

---
## 目前该引擎包含的功能(包含计划)

  - [x] 质体(粒子)
  - [x] 可以定制的质体作用力发生器
  - [x] 可以定制的区域作用力
  - [ ] 质体碰撞处理
  - [ ] 质体连杆、绳索
  - [ ] 液体模拟
  - [ ] 刚体
  - [ ] 可以定制的刚体作用力发生器
  - [ ] 刚体碰撞处理
